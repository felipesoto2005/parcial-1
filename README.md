Qué ventajas tiene en comparación con poner todo el código en un solo archivo o utilizar módulos?⠀⠀⠀

Usar clases en lugar de poner todo el código en un solo bloque permite separar responsabilidades,
mejorar la legibilidad y facilitar el mantenimiento. Cada clase tiene una función clara, lo que
hace más fácil modificar o ampliar el sistema sin romper otras partes.

¿Cómo aplicaron la Programación Orientada a Objetos en su
solución? Describan el papel de las clases creadas

Viaje: encapsula los datos de cada viaje (día, origen, destino, ruta, tiempo y costo). Representa una unidad individual de transporte.
RegistroViajes: gestiona una lista de objetos Viaje. Se encarga de agregar viajes, mostrarlos y generar un resumen semanal.
Cada clase tiene una responsabilidad clara, lo que facilita la organización, reutilización y mantenimiento del código.
